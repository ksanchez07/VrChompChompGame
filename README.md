# AR_Assignment_4

1. Open the Unity Package

2. Go to scenes and open up "VRGame" 

3. Once inside, go to Window > Packet Management > Packet Manager > + > Install package by name > paste the following: com.unity.cloud.gltfast > install. This imports the packages needed to import the custom 3d models

4. Inside Build settings, connect your specific oculues then build and run

5. When the game loads on oculus, click the meta horizon button until the view resets so the board will appear in front of you

6. Grab and release Chomp Chomp on board

7. Grab the board and tilt it around to collect all 11 stars with Chomp Chomp
